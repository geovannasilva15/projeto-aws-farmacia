# Sugestão de Protótipo para o Figma

Caso deseje complementar o projeto com um link de protótipo no Figma, a plataforma pode conter as seguintes telas:

1. Tela inicial da farmácia virtual;
2. Catálogo de produtos;
3. Detalhe do produto;
4. Formulário de pedido;
5. Confirmação de solicitação;
6. Área administrativa simples para consulta de pedidos.

## Direção visual sugerida

- Cores principais: azul, verde e branco;
- Estilo limpo e institucional;
- Cards para produtos;
- Botões arredondados;
- Layout responsivo para celular e desktop.
