# Plano de Redução de Custos

## Objetivo

Demonstrar como a adoção de serviços AWS pode reduzir custos imediatos para uma farmácia fictícia.

## Estratégias de Economia

### 1. Substituição de servidores físicos

A utilização de Amazon S3, AWS Lambda e Amazon DynamoDB reduz a necessidade de compra, instalação e manutenção de servidores locais.

### 2. Pagamento conforme o uso

Os serviços selecionados permitem iniciar com baixo consumo e crescer conforme a demanda real da aplicação.

### 3. Menos manutenção operacional

Serviços gerenciados reduzem o esforço com atualizações, monitoramento físico, disponibilidade e administração de infraestrutura.

### 4. Escalabilidade gradual

A solução pode começar pequena e aumentar a capacidade à medida que a plataforma virtual recebe mais acessos.

## Serviços e Impacto

| Serviço | Como reduz custos |
|---|---|
| Amazon S3 | Evita servidores locais para arquivos e imagens |
| AWS Lambda | Executa código sob demanda, sem servidor permanente |
| Amazon DynamoDB | Reduz administração de banco de dados e permite capacidade sob demanda |

## Conclusão

A arquitetura proposta é adequada para um projeto inicial, pois permite baixo custo de entrada, simplicidade operacional e possibilidade de crescimento futuro.
